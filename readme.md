*** CUB 3D ***

_project 42 school_

CUB 3D is a project which the main goal is to create a simple 3D raycasting engine in C language. the project takes inspiration from the classic game wolfenstein 3d, utilizing a 2D map to create a 3D view utilizing raycasting techniques. The project is designed to help students understand the fundamentals of computer graphics, game development, and the principles of raycasting.
